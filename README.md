# 📚 Repositório de Aulas Front-end

Bem-vindo(a) ao repositório de **Front-end**! Este espaço foi criado 
para armazenar materiais de estudo, exercícios e testes relacionados ao aprendizado 
e prática de **Front**.

---

## 📖 **Objetivo**
O objetivo deste repositório é oferecer um ambiente organizado para:

- Praticar HTML.
- Praticar CSS.
- Estudar conceitos fundamentais de JavaScrip.
- Realizar testes e simulações com o framework React.

---

## 🛠️ **Ferramentas/Tecnologias Utilizadas**

  - HTML
  - CSS
  - JavaScript
  - Nodejs
  - React
  - VScode


## 🚀 **Principal Objetivo**
O objetivo deste repositório é oferecer um ambiente organizado para:

- Praticar comandos na linguagem de marcação HTML.
- Treinar aplicação de style com CSSs.
- Realizar testes e simulações com projetos em REACT.
- Servir como biclioteca de ideias/códigos já testados.
